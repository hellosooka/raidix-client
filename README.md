
# Getting Started
## Install
```bash
npm install
```
## Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# General functions
- Viewing all products
- Adding product 
- Deleting product by ID 
- Change props product by ID to Database
- Sorting products by:
  - ID
  - Title
  - Date
